void Slicel::set_attribute ( std::string  attr,
std::string  val 
)

Sets an attribute in the slicel.

If the attribute is not found in the slicel it will be set in the QuarterSlicel according to the first letter of the attribute name. If the first letter doesn't correspond to a QuarterSlicel or doesn't exist in the QuarterSlicel it will be ignored.
